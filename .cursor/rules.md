# Cursor Project Rules（最高优先级）

## 项目状态
当前项目已进入 P0 / 上线前稳定阶段。

## 全局强制规则
1. 禁止任何形式的自由发挥
2. 禁止重构、优化、重排代码
3. 禁止修改 UI、样式、布局、主题
4. 禁止修改 Auth / Context / 登录流程
5. 禁止新增、删除、升级依赖
6. 禁止多文件修改（除非明确允许）
7. 禁止"顺手修复""顺便优化"

## 执行规范
- 只能执行用户明确点名的修改
- 只能修改用户点名的文件
- 必须最小 diff
- 不确定时必须停下并询问
- 不允许自作主张补代码

## 上线模式
当用户提及：上线 / 稳定 / P0 / 不要再动 UI  
自动进入 Production Mode，仅允许修 bug

✅ P0-1.1 Auth（注册 → 自动登录 → Admin 同源）已通过日志与数据库验证，禁止再修改相关逻辑。
